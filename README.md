# The Art Beat - Embed Website

A single-page tabbed website showcasing programs for The Art Beat.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Tab Navigation**: Easy switching between Home, Camps, Classes, and Workshops
- **Embeddable**: Can be embedded in iframes or other platforms
- **Clean UI**: Modern, accessible design with smooth animations

## Structure

- **Home Tab**: Overview with cards linking to each section
- **Camps Tab**: Seasonal art and pottery camps with booking links
- **Classes Tab**: Placeholder for regular classes (ready for content)
- **Workshops Tab**: Placeholder for special workshops (ready for content)

## Usage

Simply open `index.html` in a web browser, or embed it in your website:

```html
<iframe src="https://your-domain.com/index.html" width="100%" height="800px" frameborder="0"></iframe>
```

## Customization

- Update the placeholder content in Classes and Workshops tabs
- Modify colors by changing CSS variables
- Add more camps by following the existing card structure
- Adjust responsive breakpoints in the media queries

## Technologies

- Pure HTML, CSS, and JavaScript
- Google Fonts (Poppins)
- No external dependencies
